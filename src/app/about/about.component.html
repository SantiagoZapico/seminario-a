Catálogo de Libros y Favoritos

Esta aplicación web permite explorar y buscar libros de un amplio catálogo por título y autor.

Además, también tiene:

  • Búsqueda de libros de un género en específico de la API de openlibrary.org
  • Ver tu lista de libros favoritos personalizada
  • Para el ejemplo, podés buscar libros como... "Romeo and Juliet", o "A" y filtrarlos, etc...
